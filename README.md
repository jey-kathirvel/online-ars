# Akshat Royal Stay Online

FastAPI website and booking frontend for `online.akshatroyalstay.in`.

## Local setup

```powershell
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
.venv\Scripts\uvicorn app:app --reload
```

Open `http://localhost:8000`. Run tests with `.venv\Scripts\pytest`.
